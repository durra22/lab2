# lab2

Dataset
This project utilizes the Telco Customer Churn dataset, which was sourced from Kaggle. Kaggle is recognized as an approved open-data platform according to the Lab 2 guidelines.

Machine Learning Problem
The main objective of this project is to develop a model that can determine whether a customer is likely to churn by analyzing various customer attributes, including contract details, payment methods, and subscribed services.

Problem Type
The task addressed in this project is a classification problem.

Target Variable
The target variable is Churn, which represents whether a customer has discontinued the service or remains with the company.
